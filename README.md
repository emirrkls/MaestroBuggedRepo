# E-commerce Shopping Cart System

This project provides a basic shopping cart implementation for an e-commerce platform. It handles adding items, removing items, applying discounts, and calculating the total price including shipping fees.

## Features
- Add items with specific prices and quantities.
- Remove items from the cart.
- Apply percentage-based discounts to orders.
- Calculate subtotal and final total (including shipping).

## Development
To verify the system functionality and ensure all business rules are met, run the unit tests:

```bash
python test_cart.py
```
